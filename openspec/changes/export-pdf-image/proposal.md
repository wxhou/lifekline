# PDF/图片导出功能提案

## 需求确认

| 选项 | 选择 |
|------|------|
| 导出位置 | 顶部（结果页面标题旁边） |
| K线图 | 包含 |
| 水印 | 需要（"卜算子"品牌水印） |

## 功能概述

在结果页面添加"导出报告"按钮，支持导出为 PNG 图片和 PDF 文档两种格式。

## UI 设计

```
┌─────────────────────────────────────────────────┐
│  命盘分析                              [导出报告 ▼] │
│                                    ┌──────────┐ │
│  [← 重新测算]                        │ 导出PNG  │ │
│                                      │ 导出PDF  │ │
│  人生流年走势                            └──────────┘ │
│  绿涨=吉，红跌=凶 ...                             │
│                                                         │
│  [K线图 - 包含在水印区域外]                          │
│                                                         │
│  命理总评: 8/10 ...                                   │
│                                                         │
│  [事业] [财富] [婚姻] [健康] [六亲]                    │
└─────────────────────────────────────────────────┘
          底部水印: "卜算子 | 八字命理可视化"
```

## 技术方案

### 依赖库
- `html2canvas` - 将 DOM 转为 canvas
- `jspdf` - 生成 PDF 文档
- `lucide-react` - 已有导出图标

### 导出范围
1. 顶部：标题 + 导出按钮
2. K线图：完整图表（canvas 渲染）
3. 命理分析：所有评分卡片
4. 底部：水印区域

### 文件命名
- PNG: `卜算子_命理分析_{YYYYMMDD}.png`
- PDF: `卜算子_命理分析_{YYYYMMDD}.pdf`

### 水印设计
- 位置：底部居中
- 内容："卜算子 | 八字命理可视化"
- 样式：半透明浅灰色

## 修改文件

- `app/components/AnalysisResult.tsx` - 添加导出按钮和功能
- `package.json` - 添加 html2canvas, jspdf 依赖
