# 结果页面样式修复提案

## 问题分析

根据测试结果页面截图，发现以下与水墨主题不符的问题：

### 1. K线图配色问题（严重）
- **现状**：绿色代表上涨（吉），红色代表下跌（凶）- 标准股票配色
- **问题**：与页面文字说明"绿色代表吉，红色代表凶"存在逻辑混淆
- **修复**：改为传统命理配色 **红涨(吉) 绿跌(凶)**

### 2. 评分颜色问题
- **现状**：评分数字显示为蓝色（indigo）
- **问题**：蓝色看起来像超链接，与暗色水墨主题不协调
- **修复**：改为金色/琥珀色 (#c9a959)

### 3. 英文标签问题
- **现状**：使用 "RATING" 英文标签
- **修复**：改为中文"评分"

### 4. 缺少水墨装饰
- **现状**：结果页面较简洁，缺少水墨边框/纹理
- **修复**：✅ 添加与首页一致的水墨风格边框装饰

## 修改范围

- `app/components/AnalysisResult.tsx` - 修复评分颜色和标签
- `app/components/LifeKLineChart.tsx` - 改为传统命理配色（红涨绿跌）
